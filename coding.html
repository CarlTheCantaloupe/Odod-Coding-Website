Further into these three languages

Scratch

If you want to do some coding, at the top of the page, click Join Scratch, and follow the instructions for making an account. Also, on the Scratch home page, you can view other people's projects. Anyway, when you have made your account, click the icon of a folder. This will take you to My Stuff. Then, click New project. From here, you can code a program by dragging blocks into the scripts area and either clicking on your script or pressing the green flag to run it. There is a help bar on the right if you need it, as well as a Scratch Wiki. You can explore the Scratch site, and have fun!

NOTE: There is no syntax, so you can't get an error.

Python

Python is probably the next step in programmning. I highly recommend that you get comfortable with Scratch before starting with Python. When you go to the Python website, if you don't want to download the Python shell, you can still use the online console to do some stuff.

How to get to the online console

1. Go to the Python website.

2. Don't scroll or click anyhting. On the front page, there is a slideshow about Python. On that slideshow is this sign: >_. Click on it, and it will give you the online console.

NOTE: To get the >_ sign, you need to hover your mouse over the slideshow.

JavaScript

Javascript is a good language to know, as it is the language used to make everything on the web. It has quite a simple syntax.

Wait a sec, what's syntax?

Syntax is the rules about how you lay out the things in a language, like the grammar.
